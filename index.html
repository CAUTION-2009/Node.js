Node.js Негиздери

JavaScriptти серверде иштетүү платформасы

Node.js деген эмне?

Node.js — бул Chrome V8 JavaScript кыймылдаткычынын негизинде түзүлгөн, JavaScript кодун серверде иштетүүгө мүмкүнчүлүк берген ачыкулуктуу платформа.

Бул технология асинхрондуу, окуяга негизделген архитектурага ээ. Бул маанилүү себеби, Node.js бир эле учурда көптөгөн байланыштарды тейлей алат.

  • Асинхрондуу иштөө (Non-blocking I/O)
  • Жогорку ылдамдык (V8 Engine)
  • NPM — 2 млн+ пакеттер
  • Бир тил — JavaScript
  • Микросервис архитектурасы
server.js
// http модулун кошуу
const http = require('http');
// Сервер түзүү
const server = http.createServer((req, res) => {
  res.writeHead(200, { 'Content-Type': 'text/plain' });
  res.end('Салам, Node.js!');
});
// Портто угуу
server.listen(3000, () => {
  console.log('Сервер 3000-портто иштеп жатат');
});

Express.js

Node.js үчүн эң популярдуу веб-фреймворк

Орнотуу

Express.js орнотуу жана негизги сервер түзүү:

const express = require('express'); const app = express(); app.listen(3000);

Маршруттар

GET, POST жана башка HTTP методдору менен иштөө:

app.get('/', (req, res) => { res.send('Башкы бет'); });

Middleware

Аралык функциялар — запросдорду иштетүү:

app.use(express.json()); app.use((req, res, next) => { console.log(req.url); next(); });

Шаблондор

EJS/Pug шаблондору менен динамикалык баракчалар:

app.set('view engine', 'ejs'); app.get('/', (req, res) => { res.render('index', { title: 'Башкы' }); });

CORS

Кросс-домен запросторду уруксат берүү:

const cors = require('cors'); app.use(cors({ origin: 'http://localhost:3000' }));

Аутентификация

JWT токен менен колдонуучуну текшерүү:

const jwt = require('jsonwebtoken'); const token = jwt.sign( { id: user.id }, 'secret' );

Файлдар Менен Иштөө

fs (File System) модулу менен файлдарды окуу жана жазуу

fs модулу мисалдары

// Файлга жазуу
const fs = require('fs');
// Синхрондуу жазуу
fs.writeFileSync(
  'data.txt',
  'Салам, Node.js!'
);
// Асинхрондуу окуу
fs.readFile('data.txt', 'utf8',
  (err, data) => {
    if (err) console.error(err);
    else console.log(data);
  }
);
// JSON файлга жазуу
const data = { name: "Айбек", age: 25 };
fs.writeFileSync(
  'user.json',
  JSON.stringify(data, null, 2)
);

fs.writeFile()

Файлга маалымат жазуу. Эски маалыматты алмаштырат.

fs.readFile()

Файлдан маалымат окуу. Коллбэк же промис менен.

fs.appendFile()

Файлдын аягына маалымат кошуу. Эскини сактап калат.

fs.mkdir()

Жаңы папка түзүү. Рекурсивдүү режимди колдонсо болот.

fs.unlink()

Файлды өчүрүү. Кайтарылбас иш-аракет!

fs.createReadStream()

Чоң файлдарды stream аркылуу окуу. Памятьти үнөмдөйт.

REST API

REST архитектурасы менен API түзүү

GET

Маалымат алуу

Бардык колдонуучуларды же бир колдонуучуну алуу

GET /api/users
GET /api/users/:id
POST

Маалымат кошуу

Жаңы колдонуучу же ресурс түзүү

POST /api/users
Body: {name, email}
PUT

Маалымат жаңыртуу

Бардык талааларды жаңы маанилер менен алмаштыруу

PUT /api/users/:id
Body: {name, email, age}
DELETE

Маалымат өчүрүү

Колдонуучуну же ресурсту системадан өчүрүү

DELETE /api/users/:id
Status: 204 No Content
api.js — Express REST API мисалы
const express = require('express');
const app = express();
app.use(express.json());
let users = [{ id: 1, name: "Айбек" }];
// Бардык колдонуучулар
app.get('/api/users', (req, res) => {
  res.json(users);
});
// Жаңы колдонуучу
app.post('/api/users', (req, res) => {
  const user = { id: Date.now(), ...req.body };
  users.push(user);
  res.status(201).json(user);
});
app.listen(3000);
